
# AirGuard - TechnoFest 2026 Submission

**Winner: 2nd Prize | Category: Junior (Non-BAASC Schools)**

AirGuard is a drone pre-flight planning tool built in 24 hours for the **TechnoFest 2026 National Hackathon**. Our goal was to create an easy-to-use interface for drone pilots that automatically checks for safety rules, weather conditions, and restricted airspaces.
